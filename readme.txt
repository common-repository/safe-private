A Private Wordpress, a Members-only Website, helpful during a site maintenance and useful for developers...

== Description ==

The Safe Private Plugin set up a members-only area on your Wordpress and protect your content, pages and resources from public access.
Only registered users will be able to browse your site. The plugin will redirect all others users to the login page of your blog.
Deny access to unregistered users, perfect for a private website, helpful during a blog maintenance and useful for developers...

 - Membrers-only area
 - Members only feed
 - No template modifications
 - Fast, Safe and secure...

Author Plugin link page: [Safe Private](http://www.sherpah.net/wordpress/plugin-safe-private/)

== Installation ==

1. Upload the folder `safe-private` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to enable the plugin ? =

Activate the plugin through the 'Plugins' menu in WordPress

= How to disable the plugin ? =

Deactivate the plugin through the 'Plugins' menu in WordPress

= If i can't access to 'Plugins' menu in WordPress, How to disable the plugin ? =

You can remove the folder `safe-private` in the `/wp-content/plugins/` directory.
